# PWA 使用步驟（圖示已放入 public 資料夾後）

## ✅ 步驟 1：確認圖示檔案

請確認 `public` 資料夾中有以下檔案：
- `icon-192.png` (192x192 像素)
- `icon-512.png` (512x512 像素)

如果您的圖示檔案名稱不同，請重新命名為上述名稱。

---

## ✅ 步驟 2：啟動服務

開啟命令提示字元（或 PowerShell），執行：

```bash
cd c:\Users\user\Desktop\newtaipei
npm start
```

應該會看到類似這樣的訊息：
```
Server running on http://localhost:3000
```

**保持這個視窗開啟**，不要關閉。

---

## ✅ 步驟 3：在電腦瀏覽器測試

1. 開啟瀏覽器（Chrome 或 Edge）
2. 前往：`http://localhost:3000`
3. 按 `F12` 開啟開發者工具
4. 切換到「Application」或「應用程式」分頁
5. 左側選單點選「Manifest」
6. 檢查是否有錯誤

**應該看到**：
- ✅ 應用程式名稱：新北市地址查詢系統
- ✅ 圖示顯示正常
- ✅ 沒有紅色錯誤訊息

---

## ✅ 步驟 4：讓手機可以連線

### 方法 A：同一 Wi-Fi 網路（推薦）

1. **查詢電腦的 IP 位址**：
   - 開啟命令提示字元
   - 輸入：`ipconfig`
   - 找到「IPv4 位址」，例如：`192.168.1.100`

2. **確認手機和電腦連到同一個 Wi-Fi**

3. **在手機瀏覽器開啟**：
   - Android：`http://192.168.1.100:3000`（替換成您的 IP）
   - iOS：`http://192.168.1.100:3000`（替換成您的 IP）

### 方法 B：使用 ngrok（公開網址）

如果方法 A 不行，可以使用 ngrok：

1. **安裝 ngrok**：
   ```bash
   npm install -g ngrok
   ```

2. **啟動 ngrok**（開啟新的命令提示字元視窗）：
   ```bash
   ngrok http 3000
   ```

3. **複製網址**：
   - 會看到類似：`https://xxxx-xx-xx-xx-xx.ngrok.io`
   - 複製這個網址

4. **在手機瀏覽器開啟這個 ngrok 網址**

---

## ✅ 步驟 5：安裝到手機主畫面

### Android（Chrome 瀏覽器）

1. 在手機 Chrome 開啟網站
2. 點選右上角「⋮」（三個點）
3. 選擇「**安裝應用程式**」或「**新增至主畫面**」
4. 確認安裝
5. 主畫面上會出現「新北地址查詢」圖示

### iOS（Safari 瀏覽器）

1. 在手機 Safari 開啟網站
2. 點選下方中間的「分享」按鈕（方框+箭頭）
3. 向下滑動，選擇「**加入主畫面**」
4. 確認加入
5. 主畫面上會出現「新北地址查詢」圖示

---

## ✅ 步驟 6：從主畫面使用

安裝完成後：
1. 點擊主畫面上的「新北地址查詢」圖示
2. 應用程式會以全螢幕模式開啟（像原生 APP）
3. 開始使用！

---

## 🔧 疑難排解

### 問題：手機連不上電腦的服務

**解決方案**：
- 確認電腦和手機在同一 Wi-Fi
- 確認 Windows 防火牆允許 3000 埠
- 嘗試使用 ngrok

### 問題：看不到「安裝應用程式」選項

**解決方案**：
- 確認使用 HTTPS 或 localhost（ngrok 提供 HTTPS）
- 確認 manifest.json 正確載入
- 清除瀏覽器快取後重試

### 問題：圖示顯示不正確

**解決方案**：
- 確認檔案名稱正確：`icon-192.png` 和 `icon-512.png`
- 確認檔案在 `public` 資料夾中
- 清除瀏覽器快取後重試

### 問題：Service Worker 無法註冊

**解決方案**：
- 確認使用 HTTPS 或 localhost
- 檢查 `service-worker.js` 檔案是否存在
- 查看瀏覽器控制台的錯誤訊息
