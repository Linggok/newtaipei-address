# 🚀 PWA 快速開始指南

## ✅ 步驟 1：確認圖示檔案（已完成）

請確認 `public` 資料夾中有：
- ✅ `icon-192.png` (192x192 像素)
- ✅ `icon-512.png` (512x512 像素)

**如果檔案名稱不同**，請重新命名為上述名稱。

---

## 📋 步驟 2：啟動服務

### 開啟命令提示字元（PowerShell）

1. 按 `Win + R`
2. 輸入 `cmd` 或 `powershell`
3. 按 Enter

### 執行以下命令：

```bash
cd c:\Users\user\Desktop\newtaipei
npm start
```

**應該會看到**：
```
Server running on http://localhost:3000
```

**⚠️ 重要：保持這個視窗開啟，不要關閉！**

---

## 🖥️ 步驟 3：在電腦測試

1. 開啟瀏覽器（Chrome 或 Edge）
2. 前往：`http://localhost:3000`
3. 應該可以看到網站正常運作

**測試 PWA 設定**：
- 按 `F12` 開啟開發者工具
- 切換到「Application」分頁
- 左側點選「Manifest」
- 檢查是否有錯誤

---

## 📱 步驟 4：讓手機可以連線

### 方法 A：同一 Wi-Fi（最簡單）

1. **查詢電腦 IP**：
   - 在命令提示字元輸入：`ipconfig`
   - 找到「IPv4 位址」，例如：`192.168.1.100`

2. **確認手機和電腦連到同一個 Wi-Fi**

3. **在手機瀏覽器開啟**：
   ```
   http://192.168.1.100:3000
   ```
   （把 `192.168.1.100` 換成您的 IP）

### 方法 B：使用 ngrok（如果方法 A 不行）

1. **安裝 ngrok**（在新的命令提示字元視窗）：
   ```bash
   npm install -g ngrok
   ```

2. **啟動 ngrok**：
   ```bash
   ngrok http 3000
   ```

3. **複製顯示的網址**（例如：`https://xxxx.ngrok.io`）

4. **在手機瀏覽器開啟這個網址**

---

## 📲 步驟 5：安裝到手機主畫面

### Android（Chrome）

1. 在手機 Chrome 開啟網站
2. 點選右上角「⋮」（三個點）
3. 選擇「**安裝應用程式**」或「**新增至主畫面**」
4. 確認安裝

### iOS（Safari）

1. 在手機 Safari 開啟網站
2. 點選下方「分享」按鈕（方框+箭頭）
3. 選擇「**加入主畫面**」
4. 確認加入

---

## ✅ 步驟 6：完成！

安裝後，主畫面上會出現「新北地址查詢」圖示，點擊即可使用！

---

## 🔧 如果遇到問題

### 手機連不上？

- 確認電腦和手機在同一 Wi-Fi
- 確認 Windows 防火牆允許 3000 埠
- 使用 ngrok（方法 B）

### 看不到安裝選項？

- 確認使用 HTTPS（ngrok 提供）或 localhost
- 清除瀏覽器快取
- 確認 manifest.json 正確載入

### 圖示沒顯示？

- 確認檔案名稱正確：`icon-192.png` 和 `icon-512.png`
- 確認檔案在 `public` 資料夾
- 清除瀏覽器快取

---
