# 如何開啟命令提示字元並執行指令

## 方法一：從專案資料夾直接開啟（最簡單）⭐

### 步驟：

1. **開啟檔案總管**
   - 按 `Win + E` 或點選桌面上的「檔案總管」

2. **導航到專案資料夾**
   - 前往：`C:\Users\user\Desktop\newtaipei`
   - 或直接在檔案總管輸入路徑：`C:\Users\user\Desktop\newtaipei`

3. **在資料夾中開啟命令提示字元**
   - **方法 A**：在資料夾空白處，按住 `Shift` 鍵，然後**右鍵點選**
   - 選擇「**在此處開啟 PowerShell 視窗**」或「**在此處開啟命令視窗**」
   
   - **方法 B**：在資料夾上方的路徑欄位，輸入 `cmd` 然後按 Enter
   
   - **方法 C**：在資料夾上方的路徑欄位，輸入 `powershell` 然後按 Enter

4. **確認位置正確**
   - 命令提示字元視窗應該顯示：
   ```
   C:\Users\user\Desktop\newtaipei>
   ```

---

## 方法二：從開始功能表開啟

### 步驟：

1. **開啟命令提示字元**
   - 按 `Win + R`
   - 輸入 `cmd`
   - 按 Enter

2. **導航到專案資料夾**
   - 在命令提示字元中輸入：
   ```bash
   cd C:\Users\user\Desktop\newtaipei
   ```
   - 按 Enter

3. **確認位置正確**
   - 應該看到：
   ```
   C:\Users\user\Desktop\newtaipei>
   ```

---

## 方法三：使用 PowerShell（Windows 10/11）

### 步驟：

1. **開啟 PowerShell**
   - 按 `Win + X`
   - 選擇「Windows PowerShell」或「終端機」
   - 或按 `Win + R`，輸入 `powershell`，按 Enter

2. **導航到專案資料夾**
   ```powershell
   cd C:\Users\user\Desktop\newtaipei
   ```

3. **確認位置正確**
   - 應該看到：
   ```
   PS C:\Users\user\Desktop\newtaipei>
   ```

---

## 確認您在正確的位置

執行以下指令確認：

```bash
dir
```

或

```powershell
Get-ChildItem
```

**應該看到**：
- `server.js`
- `package.json`
- `public` 資料夾
- `*.csv` 檔案
- 等等...

如果看到這些檔案，表示位置正確！

## 執行 Git 指令

確認位置正確後，就可以執行 Git 指令了：

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的帳號/newtaipei-address.git
git push -u origin main
```

## 常見問題

### Q: 找不到「在此處開啟 PowerShell 視窗」選項？

**A**: 
- 確認按住 `Shift` 鍵
- 或在路徑欄位輸入 `powershell` 按 Enter

### Q: 命令提示字元顯示的路徑不對？

**A**: 
- 使用 `cd` 指令切換到正確位置：
  ```bash
  cd C:\Users\user\Desktop\newtaipei
  ```

### Q: 輸入指令後沒有反應？

**A**: 
- 確認按 Enter 執行
- 確認在正確的資料夾中
- 檢查是否有錯誤訊息
