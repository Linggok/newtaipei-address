# 🚀 快速部署指南 - 不需要電腦開著

## 問題

**目前**：需要電腦開著，手機才能使用  
**解決**：部署到雲端，手機隨時可以存取

---

## 🎯 最簡單的方案：Railway（推薦）

### 為什麼選 Railway？
- ✅ 免費額度足夠個人使用
- ✅ 支援 Node.js 後端（您的 server.js）
- ✅ 設定簡單
- ✅ 自動部署

### 步驟（5 分鐘完成）

#### 1. 準備 GitHub 專案

如果還沒有 GitHub 帳號：
1. 前往 https://github.com 註冊

將專案上傳到 GitHub：

```bash
# 在專案資料夾執行
git init
git add .
git commit -m "Initial commit"
git branch -M main

# 在 GitHub 建立新專案後，執行：
git remote add origin https://github.com/你的帳號/newtaipei-address.git
git push -u origin main
```

#### 2. 部署到 Railway

1. **前往 Railway**：https://railway.app
2. **登入**：使用 GitHub 帳號登入
3. **建立新專案**：
   - 點選「New Project」
   - 選擇「Deploy from GitHub repo」
   - 選擇您的專案
4. **等待部署**：Railway 會自動偵測並部署
5. **取得網址**：部署完成後會顯示網址，例如：
   ```
   https://newtaipei-address.up.railway.app
   ```

#### 3. 完成！

現在您的手機可以：
- ✅ 隨時開啟這個網址
- ✅ 安裝到主畫面
- ✅ 不需要電腦開著

---

## 🎯 替代方案：Render（也免費）

### 步驟

1. **前往 Render**：https://render.com
2. **登入**：使用 GitHub 帳號
3. **建立 Web Service**：
   - 點選「New +」→「Web Service」
   - 選擇您的 GitHub 專案
   - 設定：
     - Name: `newtaipei-address`
     - Environment: `Node`
     - Build Command: `npm install`
     - Start Command: `npm start`
     - Plan: **Free**
4. **部署**：點選「Create Web Service」
5. **取得網址**：部署完成後會顯示網址

---

## 📱 部署後的手機使用

1. **在手機瀏覽器開啟雲端網址**
2. **安裝 PWA**：
   - Android：選單 → 安裝應用程式
   - iOS：分享 → 加入主畫面
3. **完成！** 隨時可以使用，不需要電腦開著
